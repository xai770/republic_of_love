> _Clarity without pressure. Warmth without cuteness. Modernity without trend-chasing._

## 2. Shape Language

### 2.1 Geometry Rules

Allowed shapes:

- Circles
    
- Rounded rectangles
    
- Rounded squares
    
- Soft triangles
    
- Organic blobs (controlled)
    

Forbidden:

- Sharp corners
    
- Thin spikes
    
- Aggressive diagonals
    
- Overlapping sharp geometry
    

All corners must be **visibly rounded**.

---

### 2.2 Complexity Rule

- **One icon = one idea**
    
- No visual subplots
    
- No hidden metaphors
    
- No â€œcleverâ€ symbolism
    

If an icon needs explanation, it fails.

## 4. Depth & Lighting

### 4.1 Depth Philosophy

Icons must feel **tactile**, not flat â€” but never heavy.

Allowed:

- Light highlights
    
- Soft tonal separation
    

Forbidden:

- Drop shadows
    
- Harsh outlines
    
- Glassmorphism
    
- Glow effects
    

Depth comes from **light**, not shadows.

## 6. Size & Responsiveness

Icons must work at:

- 16px (minimum)
    
- 20px
    
- 24px
    
- 32px

- in low contrast
