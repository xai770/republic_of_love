## Implementation

### Frontend Framework Decision

| Option | Pros | Cons |
|--------|------|------|
| **Plain HTML** | Simple, fast | Manual state management |
| **React** | Rich ecosystem | Heavy for our needs |
| **Vue** | Lightweight, reactive | Learning curve |
| **Svelte** | Minimal JS, fast | Smaller ecosystem |
| **HTMX** | Server-rendered, minimal JS | Limited interactivity |

**Recommendation:** Start with HTMX + Jinja2 templates. Server-rendered, minimal JavaScript, good enough for MVP.
