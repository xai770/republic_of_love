## Sandy's Notes

**Work history is the source of truth.** Skills are derived. Don't let users edit skills directly â€” they'd game it.

**The description field is where value lives.** Coach users: "Include numbers, technologies, outcomes."

**Async extraction is important.** Don't make users wait for LLM processing after every keystroke. Debounce or trigger on explicit save.

**Multi-method is worth the extra 2h.** User choice reduces friction. Some people hate forms. Some people don't have an updated CV. Meet them where they are.

**Upload is actually the quickest win.** PyMuPDF already extracts text. LLM parses to structured. Users with recent CVs are onboarded in 60 seconds.
